立项报告：留学生信息管理平台

项目简介

名称：留学生信息管理平台

主持人：待定小组

项目背景与意义

本项目来源于实际需求，经调研，国际教育学院的留学生管理方式存在着诸多不便。留学生的报名申请、身份登记、信息管理依旧依赖人工录入，复杂的审核流程也都依赖于纸质表格。并且留学生在校的监管方式也十分的麻烦。例如，留学生的位置信息徐定时上报给公安部门，然而负责人统计留学生位置时，却要手工统计大家在微信中发送的位置信息，相当不方便。

因此考虑开发一个留学生信息管理平台，切实解决国际教育学院留学生信息管理及在校留学生监管问题。

项目用户

  • 留学生：

    • web端，在线申报专业，在线支付，申报进度查询与结果通知。
    • 手机端，个人身份信息的修改，请假，考勤，通报位置。

  • 学校管理员：

    • web端，留学生入学报名申报信息的审核，留学生日常信息的监管与奖学金积分评比制度。
    • 手机端，能根据留学生姓名或者签证等信息查询到留学生的详细信息。

项目功能

手机端

个人信息管理

  • 个人信息的录入与更新，包括中英文姓名、国籍、公寓基本等信息
  • 签证到期提醒功能，奖学金积分展示与通知

考勤

  • 宿舍考勤、课程考勤功能
  • 活动扫描二维码签到、报名等

消息推送

  • 留学生政策、校务通知、活动资讯等推送
  • 奖学金积分变动公布

web端

留学生在线申报

  • 留学生报名
  • 留学生审核，主要由老师或学院负责人进行流程审核

留学生信息监管

  • 留学生身份信息管理
  • 签到、考勤等后台管理

项目实现方案

系统架构

平台分两个模块，分别是留学生在线报名模块和在校留学生在校监管模块。初步打算将其做成两个系统，只在数据库层面进行共享。前者是BS架构，以浏览器使用为主，部分界面应考虑移动端兼容。后者为CS架构，以手机端使用为主。

技术架构

  • 后台采用Jeesite开源框架，集成了CMS内容管理、OA系统和基于Acticity工作流引擎的流程管理。便于进行报名流程的管理。该平台的技术栈如下，留学生信息管理平台技术栈
  • 前端网页采用amaze UI框架
  • 手机端采用appcan跨平台快速开发框架，便于进行快速跨平台开发。

项目学习价值

  • 该平台根据实际需求出发，并且涉及当前主流Java企业级开发框架。以及Activity工作流引擎技术，便于提高开发技术和素养。
  • appcan是有名的跨平台开发技术，可快速便捷的创建不那么复杂的企业级应用，也特别方便用于创业初期，资金紧张的情况。
  • Native React(可选)

可行性分析

  • 该项目来源与实际需求，并且方便与真正的客户（与国教学院老师较为熟悉）进行沟通交流。
  • 小组技术负责人曾做过相类似项目，乐学（应用宝市场可查，appstore名称为乐学中国），有类似项目的开发经验。
  • 国际教育学院老师愿意提供测试环境，可提供数名学生方便项目前期需求调研以及后期项目测试。
  • 考虑工作量可能过大，可根据前期需求调研阶段结果调整为偏向在校留学生监管功能的开发。

难点与风险

  • 小组人员不够，需要前段和设计人员
  • Activity工作流引擎不熟悉
  • 工作量大

后台管理模块技术选型

1、后端

  • 核心框架：Spring Framework 4.1（掌握）
  • 安全框架：Apache Shiro 1.2（掌握）
  • 视图框架：Spring MVC 4.1（掌握）
  • 服务端验证：Hibernate Validator 5.2（熟悉）
  • 布局框架：SiteMesh 2.4
  • 工作流引擎：Activiti 5.21（掌握）
  • 任务调度：Spring Task 4.1
  • 持久层框架：MyBatis 3.2（掌握）
  • 数据库连接池：Alibaba Druid 1.0 （熟悉）
  • 缓存框架：Ehcache 2.6、Redis
  • 日志管理：SLF4J 1.7、Log4j
  • 工具类：Apache Commons、Jackson 2.2、Xstream 1.4、Dozer 5.3、POI 3.9（熟悉）

2、前端(了解)

  • JS框架：jQuery 1.9。
  • CSS框架：Twitter Bootstrap 2.3.1（稳定是后台，UI方面根据需求自己升级改造吧）。
  • 客户端验证：JQuery Validation Plugin 1.11。
  • 富文本在线编辑：CKEditor
  • 在线文件管理：CKFinder
  • 动态页签：Jerichotab
  • 手机端框架：Jingle
  • 数据表格：jqGrid
  • 对话框：jQuery jBox
  • 下拉选择框：jQuery Select2
  • 树结构控件：jQuery zTree
  • 日期控件： My97DatePicker

4、平台

  • mac/windows
  • 服务器中间件：在Java EE 5规范（Servlet 2.5、JSP 2.1）下开发，支持应用服务器中间件
    有Tomcat 6+、Jboss 7+、WebLogic 10+、WebSphere 8+。
  • 数据库支持：目前仅提供MySql和Oracle数据库的支持，但不限于数据库，平台留有其它数据库支持接口，
    你可以很方便的更改为其它数据库，如：SqlServer 2008、MySql 5.5、H2等

5、工具&环境

  • IntelliJ Idea 15
  • Navicat
  • putty,winscp or Filezilla(windows,mac不需要)
  • chrome、IE浏览器（必须，学校办公环境要求必须支持ie）
  • Java、Maven、git
